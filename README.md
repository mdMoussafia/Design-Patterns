# Design Patterns


## 📖 Design Patterns inclus

### 1️⃣ DP-Adapter
Permet de faire collaborer des interfaces incompatibles en les adaptant.

### 2️⃣ DP-Composite
Permet de traiter des objets individuels et des collections d’objets de manière uniforme.

### 3️⃣ DP-Decorator
Permet d’ajouter dynamiquement des fonctionnalités à un objet sans modifier son code.

### 4️⃣ DP-Observer
Permet de notifier automatiquement des objets abonnés lors d’un changement d’état.

### 5️⃣ DP-Proxy
Permet de contrôler l’accès à un objet, pour la sécurité ou l’optimisation des ressources.

### 6️⃣ DP-Strategy
Permet de définir une famille d’algorithmes et de les rendre interchangeables à l’exécution.
